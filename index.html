Projects


Parsing of NASA Thermodynamic data

NASA came up with polynomials that can be used to evaluate the thermodynamic properties such as Enthalpy, Entropy and specific heat for various gases using this data. A program was written to be able to parse the different high and low temperature polynomials and calculate the required properties and plot the graphs for the various gases.

Gesture Controlled Robot

Using an Arduino uno board and an accelerometer and a gyroscope sensor along with radio transmission modules a car was created that works on the gestures of the hand. Predefined gestures were programmed and the car was made to react to these gestures.




Electricity Generating shoes

The project was done by the principle of piezoelectric effect, a dozen of piezoelectric sensors that were connected together and placed under the sole of the shoe so that when we move around the piezoelectric sensors get pressed and electricity is generated and stored in a battery.
